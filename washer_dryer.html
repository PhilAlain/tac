Washer & Dryer

Washing Machine short instructions.

Load the Machine

Open the washing machine door and place clothes inside.

Ensure all pockets are empty :-)

Do not overfill; leave enough space for clothes to move freely.

Detergent compartments

Add Detergent

Pour the appropriate amount of detergent into the detergent drawer II..

Use fabric softener if desired in the middle compartment*.

Cycle selection & start button

Select Cycle

Choose the appropriate wash cycle based on the fabric type..

Rapid 45' is a good starting point.

Select Temperature

Depending on the selected cycle, the Temperature button proposes several possibilities..

This step is not mandatory.

Start the machine

Close the door and press the Start button..

The machine will play music to notify you when the cycle is finished.
