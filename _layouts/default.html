To ensure the styles of the `jekyll-theme-cayman` theme are included directly in your `_layouts/default.html`, you can manually copy the CSS styles from the theme into a `

{{ site.title }}

{{ site.description }}

{{ content }}
This page was generated by GitHub Pages.
``` This will apply the styles directly within your HTML file, bypassing the need for an external stylesheet. Adjust the CSS styles as needed to match the theme's original appearance.
